#### Update `dao.json`



```json title="Configure your DAO Addresses in /config/dao.json"
{
	"daoAddress": "YOUR_DAO_ADDRESS",
	"tokenManagerAddress": "YOUR_TOKEN_MANAGER_ADDRESS",
	"votingAddress": "YOUR_VOTING_ADDRESS"
}
```
