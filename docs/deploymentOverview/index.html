AraCred Manual Deployment

Testing.

This is how to deploy your own AraCred instance on manual mode.

#Prerequisites

  • A computer connected to the internet.
  • An Ethereum address with some ETH.
  • An Aragon DAO. (You'll need at least 0.1 ETH to deploy your own DAO)
  • A GitHub account.
  • An Infura account.

#Step 0: Basic setup (Required)

  • Fork AraCred
  • Enable GitHub Actions
  • Export Private Key
  • Infura

#Step 2: Connect GitHub (Optional)

If you don't want to connect AraCred to GitHub you'll have to remove all GitHub related info from project.json.

  • Open project.json
  • Add GitHub Repos
  • Add GitHub Usernames

#Step 3: Connect Discord (Optional)

If you don't want to connect AraCred to Discord you'll have to remove all GitHub related info from project.json.

  • Enable Developer Mode
  • Build A Bot
  • Connect Bot to GitHub
  • Connect Discord to AraCred
  • Add Discord UserIDs to AraCred
